## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Solana wallet (Phantom, Solflare, etc.)

### Installation

```bash
# Clone repository
git clone https://github.com/minerva-id/purify-protocol.git

# Navigate to project
```
